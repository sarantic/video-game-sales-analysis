# Video Game Sales Analysis

An exploratory data analysis project examining over 1,000 video games released between 2004 and 2010. The analysis investigates relationships between game ratings, sales performance, and playtime through data cleaning, statistical summaries, visualization, and bootstrap sampling.

## Project Overview

Understanding the factors associated with commercially successful video games is valuable for developers, publishers, and market analysts. This project explores historical video game data to identify patterns in sales, ESRB ratings, and gameplay length while demonstrating a complete exploratory data analysis workflow in Python.

## Features

- Exploratory analysis of a video game sales dataset
- Data cleaning and preprocessing
- Descriptive statistics
- Distribution and comparative visualizations
- Bootstrap sampling for estimating population proportions
- Interpretation of relationships between sales, ratings, and playtime

## Technologies

- pandas
- NumPy
- Matplotlib
- Jupyter Notebook

The analysis explores relationships among variables such as:
